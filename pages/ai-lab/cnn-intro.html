01

Apa itu Convolutional Neural Network?

Jaringan saraf yang dirancang khusus untuk memproses data berbentuk grid — seperti gambar

Bayangkan kamu melihat foto Tugu Yogyakarta. Otak kamu tidak memproses setiap piksel satu per satu — melainkan mendeteksi pola lokal: sudut, tepi, lengkungan, lalu menggabungkannya menjadi bentuk, lalu objek. CNN bekerja dengan cara yang sama.
Tugu Yogyakarta
📍 Yogyakarta, Indonesia
CNN melihat: tepi vertikal tiang, gradien langit, pola bundaran jalan. Layer dalam mengenali struktur monumen secara keseluruhan.
Patung Suro & Boyo
📍 Surabaya, Indonesia
CNN melihat: tekstur sisik buaya, kontur hiu, latar langit biru. Kombinasi fitur ini unik dan mudah diklasifikasi.
💡
CNN adalah arsitektur yang translation-invariant — Tugu Yogyakarta akan tetap terdeteksi meski ada di pojok kiri, tengah, atau kanan gambar, karena filter yang sama di-slide ke seluruh bagian gambar.
❌ Fully Connected (MLP)

Setiap piksel terhubung ke setiap neuron. Untuk gambar 224×224×3 = 150,528 parameter per neuron. Tidak scalable, tidak memahami struktur spasial.

VS
✅ Convolutional (CNN)

Filter kecil 3×3 = hanya 9 parameter, di-share ke seluruh gambar. Efisien, spatial-aware, dan menangkap fitur lokal.

02

Convolution Layer

Filter yang "geser" ke seluruh gambar untuk mendeteksi fitur lokal

Sebuah filter (kernel) adalah matriks kecil berisi angka-angka yang dipelajari selama training. Filter ini digeser (slide) ke seluruh gambar — di setiap posisi, dilakukan perkalian elemen-per-elemen lalu dijumlahkan. Hasilnya disebut feature map.

📋 Langkah demi Langkah: Satu Sliding Window
1
Posisikan filter

Filter 3×3 diletakkan di pojok kiri atas gambar, menutupi 9 piksel pertama.

2
Element-wise multiplication

Setiap piksel di bawah filter dikalikan dengan nilai kernel yang bersesuaian.

3
Sum → satu nilai output

Semua 9 hasil perkalian dijumlahkan menjadi satu angka di feature map.

4
Geser (stride) ke kanan

Filter bergeser 1 piksel ke kanan, proses diulang. Setelah sampai ujung kanan, turun 1 baris.

PyTorch · Conv2d
import torch
import torch.nn as nn

# Single conv layer: 3 input channels (RGB), 32 output channels, 3×3 kernel
conv = nn.Conv2d(in_channels=3, out_channels=32,
                  kernel_size=3, stride=1, padding=1)

# Input: batch of 4 images, 3 channels, 224×224
x = torch.randn(4, 3, 224, 224)
out = conv(x)
print(out.shape)   # torch.Size([4, 32, 224, 224])
#                      ↑    ↑     ↑    ↑
#                    batch ch    H    W  (same size karena padding=1)
03

Stride & Padding

Mengontrol seberapa besar output dan seberapa cepat filter bergerak

Stride adalah jumlah langkah yang diambil filter setiap bergeser. Padding adalah penambahan border nol di sekitar input agar output tidak mengecil atau agar filter bisa menutup tepi gambar.
🎛️ Interaktif · Lihat efek stride dan padding pada ukuran output

📦
padding="same"

Menambah nol di tepi gambar agar output size = input size. Digunakan agar tidak kehilangan informasi di border. Formula: padding = (kernel_size − 1) / 2

nn.Conv2d(..., padding=1) # untuk kernel 3×3
⚡
stride=2

Filter melompat 2 piksel setiap langkah, output jadi setengah ukuran input. Alternatif lebih efisien dari Max Pooling untuk downsampling.

nn.Conv2d(..., stride=2) # output = input / 2
04

Pooling

Merangkum feature map menjadi lebih kecil — membuang detail, menjaga informasi penting

Setelah conv layer, biasanya ada pooling layer untuk memperkecil ukuran feature map. Ini membuat model lebih cepat, lebih hemat memori, dan memberikan sedikit translational invariance tambahan.

05

Apa yang dipelajari tiap layer?

Layer dangkal = fitur sederhana. Layer dalam = konsep abstrak.

Inilah yang membuat CNN luar biasa: setiap layer membangun di atas layer sebelumnya. Tidak ada yang hardcoded — semua dipelajari dari data.
