Structuring Element

The "kernel" of morphological operations — shape defines behavior

Morphological operations work on binary images (black/white). The structuring element (SE) is a small shape — like a kernel in convolution — that defines which neighbors are examined. But instead of multiplying, morphology uses set logic: does the SE fit entirely inside the object? (erosion) Does it touch at all? (dilation).
🧱
Structuring element shapes: Rectangle, ellipse, and cross are the most common. Shape matters — a cross SE preserves diagonal detail better than a rectangle. A larger SE = stronger effect.

Erosion & Dilation

The two fundamental building blocks — shrink or grow

Erosion moves the SE over every pixel. Output = 1 only if the SE fits entirely within the white region. Result: objects shrink, thin protrusions disappear, touching objects separate. Dilation is the opposite: output = 1 if the SE touches the white region at all. Result: objects grow, holes fill, broken parts reconnect.

SE must fit entirely inside white region. Shrinks objects, removes small noise, separates touching objects.

SE touches white region at any point. Grows objects, fills small holes, reconnects broken parts.
cv2.erode / cv2.dilate
gray   = cv2.imread('mask.png', 0)
# Threshold to binary first
_, binary = cv2.threshold(gray, 127, 255,
                           cv2.THRESH_BINARY)

se = cv2.getStructuringElement(
    cv2.MORPH_RECT, (5, 5))

eroded  = cv2.erode(binary, se,
                    iterations=1)
dilated = cv2.dilate(binary, se,
                     iterations=2)
📐 Erosion rule

Output pixel = 1 if every pixel covered by SE = 1.
Output pixel = 0 if any pixel covered by SE = 0.

Dilation rule:
Output pixel = 1 if any pixel covered by SE = 1.

Dual relationship: Erosion of A by SE = complement of (dilation of complement(A) by SE).

Opening & Closing

          
Combining erosion + dilation for noise removal and hole filling

        

        
Opening = Erode → Dilate. The erosion removes small white noise and thin protrusions; the dilation restores the main object size. Net effect: noise is gone, objects unchanged. Closing = Dilate → Erode. The dilation bridges small gaps and fills holes; the erosion restores size. Net effect: holes are filled, objects unchanged.


        
💡
Memory trick: Opening opens up gaps between objects and removes small white blobs. Closing closes holes inside objects and small black gaps.

cv2.morphologyEx — all-in-one


# Opening: erode → dilate (removes white noise)
opened = cv2.morphologyEx(binary, cv2.MORPH_OPEN,  se)

# Closing: dilate → erode (fills black holes)
closed = cv2.morphologyEx(binary, cv2.MORPH_CLOSE, se)

# Equivalent manual versions:
opened_manual = cv2.dilate(cv2.erode(binary, se), se)
closed_manual = cv2.erode(cv2.dilate(binary, se), se)

# Rule of thumb for SE size:
# SE must be LARGER than noise to remove it
# SE must be SMALLER than objects to preserve them

Skeletonization & Morphological Reconstruction

          
Reduce to 1-pixel backbone · Precision object extraction

        

        
Skeletonization repeatedly erodes an object until only a 1-pixel-wide skeleton remains — the topological center line. Useful for shape analysis, OCR, measuring vessel widths. Morphological Reconstruction uses a marker + mask to selectively restore or remove objects — perfect for removing objects touching a border or extracting only connected components of specific size.

Morphological Reconstruction


from skimage.morphology import (
    reconstruction)

# Remove border-touching objects:
marker = img.copy()
marker[1:-1, 1:-1] = 0  # only border
recon  = reconstruction(marker, img)
result = img - recon.astype(np.uint8)

# Fill holes (invert, reconstruct, invert):
seed = np.copy(img)
seed[1:-1, 1:-1] = 255
filled = reconstruction(
    255-seed, 255-img, method='erosion')
result = 255 - filled.astype(np.uint8)
